static routing
Formula 
: Destination : Submask : Nexthop/Interface

!@R1
conf t
ip route 10.1.1.5 255.255.255.255 10.1.1.2
end

ping 10.1.1.5
it will reply due to static routing


Configure Static Host route on R2
destined for R3's e1/2 interface


!@R2

end
